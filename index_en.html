Subscription Manager (JA)

Centralize team subscriptions in M365 (SharePoint list + Power Automate). Auto-emails the registered address on each billing day and rolls the next-billing date forward. Handles mixed JPY / USD / EUR with weekly FX updates. Setup guide is Japanese-only for now.

Work-hour calculator (JA)

Drop an SRKINMU CSV to tally monthly work hours. Supports configurable standard hours, pending-leave adjustments, and half-day leave (AM = 3:30 / PM = 4:15, full-time only). No AI or API — everything runs in the browser. Japanese UI only.
