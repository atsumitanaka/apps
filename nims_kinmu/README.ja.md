# NIMSジュニア勤務表ジェネレータ

NIMSジュニア研究員（Master / Doctor プログラム）の年間勤務日をカレンダー上で組み立てるWebツール。ブラウザローカル動作、単一HTML、保存不要。

**公開URL:** https://atsumitanaka.github.io/apps/nims_kinmu/

## 概要

- **対象:** Master プログラム（月11日 / 年132日）、Doctor プログラム（月14日 / 年168日）
- **原則:** 「任意の連続7日間でちょうど3日勤務」
  - 数学的にこれは週の固定パターン（曜日3つを指定）に等価
- **調整月:** 2月・3月・8月・9月とその前後7日間は上記ルール対象外
  - GW・お盆・年末年始・年度替わりを吸収するための緩衝期間
- **夏季休暇:** NIMS推奨のお盆期間（8/13〜8/16をコアに、週末・祝日を除いた平日4日）を自動で非勤務日として提示

## 使い方

1. **プログラム**（Master / Doctor）を選択
2. **開始年月**を選択（デフォルト 4月）
3. **週パターン**で勤務曜日を3つ選択（デフォルト 月・水・金）
4. カレンダー上で日付をクリックすると勤務／非勤務が切り替わる
5. 上部の年間サマリで目標値との差分を確認
6. 「CSV出力」で Excel/Numbers 用の CSV をダウンロード

## 表示ルール

| 色 | 意味 |
|---|---|
| 青 | 勤務日 |
| 薄黄 | 調整期間（自由設定可） |
| 薄紫 | 夏季休暇（お盆） |
| 白 | 非勤務日 |
| 赤枠+`!` | 週3日ルール違反（祝日・夏季休暇を含む週は判定対象外） |

## 技術メモ

- 単一 HTML ファイル、ビルド不要、依存ライブラリなし
- localStorage で編集内容を自動保存
- 日本の祝日は 1980〜2099 年で有効な近似式で算出（春分・秋分・振替休日含む）
