サブスクリプション管理システム

M365 (SharePoint + Power Automate) で契約サブスクを一元管理・請求日に自動通知

このシステムの目的

チームで契約している各種サブスクリプション（Claude, Netflix, Adobe CC 等）を SharePoint リストで一元管理し、 請求当日に契約時登録アドレスへ自動で通知メールを送るシステムです。 クレジットカード明細を毎回確認しなくても、請求のタイミングと金額が把握できます。

ポイント：導入したチーム内でしか見えません。SharePoint の権限機能で閲覧・編集を制御。
汎用性：NIMS 固有設定なし。同じ手順で他機関にも展開可能。

利用者新しいサブスクを追加する

  1. SharePoint サイト「サブスクリプション管理」を開く（管理者から共有されたリンク）
  2. 右上の 「＋ 新規」 をクリック
  3. フォームに入力：
    • サービス名：Claude Pro など
    • 月額 / 通貨：課金額と通貨（JPY / USD / EUR）
    • 課金周期：月次 / 年次 / 四半期
    • 次回請求日：次に請求される日付
    • 通知先メール：契約時に登録しているアドレス
    • 契約者：自分を選択
    • 解約URL：サービスの解約/設定ページの URL（任意）
  4. 「保存」 をクリック
翌朝 9:00 以降、次回請求日が今日と一致する日に自動でメールが届きます。 通知が来ない場合は 有効 が「はい」になっているか確認してください。

利用者解約したとき

該当行を開き、有効 を「いいえ」に変更して保存します。
履歴として残るので、後から契約履歴の振り返りができます。完全に消したい場合のみ行を削除してください。

技術スタック

  • データ層：SharePoint Online リスト
  • フロー：Power Automate（スケジュール トリガー × 2）
  • 通知：Outlook「メールの送信 (V2)」
  • 為替：exchangerate.host（週次で自動更新、Premium 不可の環境では手動更新）

外部サービス（Zapier, Airtable 等）を一切使わず、M365 のライセンス内で完結します。
