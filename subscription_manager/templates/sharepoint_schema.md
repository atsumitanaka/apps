# SharePoint リスト列定義（コピペ用）

## リスト 1: `Subscriptions`

| # | 表示名 | 内部名 | 型 | 詳細設定 | 必須 |
|---|-------|-------|-----|---------|------|
| 1 | サービス名 | Title | 1行テキスト | (Title を流用) | ✓ |
| 2 | 月額 | Amount | 通貨 | 小数点第2位、既定=0 | ✓ |
| 3 | 通貨 | Currency | 選択肢 | JPY / USD / EUR、既定=JPY、単一選択 | ✓ |
| 4 | 課金周期 | BillingCycle | 選択肢 | 月次 / 年次 / 四半期、既定=月次、単一選択 | ✓ |
| 5 | 次回請求日 | NextBillingDate | 日付と時刻 | **日付のみ** | ✓ |
| 6 | 通知先メール | NotifyEmail | 1行テキスト | メールアドレス | ✓ |
| 7 | 通知先CC | NotifyCc | 1行テキスト | `;` 区切りで複数可 |  |
| 8 | カテゴリ | Category | 選択肢 | AI / 動画配信 / 業務ソフト / クラウド / その他 |  |
| 9 | 契約者 | Owner | ユーザー | 単一選択 | ✓ |
| 10 | メモ | Notes | 複数行テキスト | プレーンテキスト、6行 |  |
| 11 | 解約URL | CancelUrl | ハイパーリンク | URL |  |
| 12 | 有効 | Active | はい/いいえ | 既定=はい | ✓ |

## リスト 2: `ExchangeRates`

| # | 表示名 | 内部名 | 型 | 詳細設定 |
|---|-------|-------|-----|---------|
| 1 | 通貨コード | Title | 1行テキスト | USD, EUR など |
| 2 | JPYレート | Rate | 数値 | 小数点第4位 |
| 3 | 最終更新 | LastUpdated | 日付と時刻 | 日付と時刻 |

初期データ：
- `USD` / `157.0` / (今日の日付)
- `EUR` / `168.0` / (今日の日付)

## 列を英語内部名で作るコツ

SharePoint は「列を作成した時点の名前」を**内部名**として保持し、その後リネームしても内部名は変わりません。Power Automate のフィルタや動的コンテンツ参照は**内部名**を使うので、以下の順で作成すると安定します：

1. まず英語で列作成（例：`Amount`）
2. 作成後に「列の設定」→「名前の変更」で日本語に変更（例：`月額`）
3. これで表示は「月額」、内部名は `Amount` になる
