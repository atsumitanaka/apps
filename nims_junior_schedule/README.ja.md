# NIMSジュニア勤務表ジェネレータ

NIMSジュニア研究員（Master / Doctor プログラム）の年間勤務日をカレンダー上で組み立てるWebツール。管理者（事務局）が複数学生の勤務表を作成し、共有リンク or メールで学生へ配布します。

**公開URL**
- 管理画面: https://atsumitanaka.github.io/apps/nims_junior_schedule/admin/
- 学生ビュー: https://atsumitanaka.github.io/apps/nims_junior_schedule/#s=... （管理者がリンク生成）

## 概要

- **対象:** Master プログラム（月11日 / 年132日）、Doctor プログラム（月14日 / 年168日）
- **原則:** 「任意の連続7日間でちょうど3日勤務」
- **調整月:** 2月・3月・8月・9月 + 前後7日間バッファ（ルール緩和）
- **夏季休暇:** NIMS推奨のお盆期間（8/13-16をコアに平日4日）を自動非勤務

## 使い方（管理者）

1. `/admin/` にアクセス
2. `+ 学生を追加` で新規学生タブを作成
3. **名前・メール** を入力、**プログラム・開始年月・週パターン** を設定
4. カレンダー日付をクリックして調整（月間・年間合計が即時反映）
5. `メールで送信` を押すとメーラーが開き、宛先・件名・共有URL入り本文が自動入力される
6. または `リンクをコピー` でチャット等に貼付

- 学生ごとに独立した設定・カレンダーが保存されます（ブラウザの localStorage）
- タブをクリックして学生切替、名前を変更するとタブラベルも即更新

## 使い方（学生）

1. 事務局から受け取ったURLを PC/スマホで開くだけ
2. 自分のカレンダー・週パターン・月別集計が表示される
3. `CSV出力` で Excel/Numbers 用データをダウンロード可
4. 編集はできません（管理者側で更新後、新しいリンクを再送してもらう運用）

## 表示ルール

| 色 | 意味 |
|---|---|
| 青 | 勤務日 |
| 薄黄 | 調整期間（自由設定可） |
| 薄紫 | 夏季休暇（お盆） |
| 白 | 非勤務日 |
| 赤枠+`!` | 週3日ルール違反（祝日・夏季休暇を含む週は対象外） |

## 技術メモ

- **単一HTML** × 4ファイル（管理JP/EN、学生JP/EN）、ビルド不要、依存なし
- **状態管理**: 管理側は localStorage（`nims_junior_admin_v1`）、学生側は URL fragment `#s=<base64>`
- **メール送信**: `mailto:` プロトコルで管理者のメーラーを起動（Outlook / Apple Mail / Gmail 等）
- **祝日**: 1980〜2099年で有効な近似式（春分・秋分・振替休日を含む）
