# NIMS Junior Work Schedule Generator

A web tool for building yearly work-day schedules for NIMS Junior researchers (Master / Doctor programs). The administrator creates schedules for multiple students and distributes them via share links or email.

**Live URLs**
- Admin: https://atsumitanaka.github.io/apps/nims_junior_schedule/admin/index_en.html
- Student view: https://atsumitanaka.github.io/apps/nims_junior_schedule/index_en.html#s=... (link generated by admin)

## Overview

- **Programs:** Master (11 days/mo, 132/yr), Doctor (14 days/mo, 168/yr)
- **Core rule:** *Exactly 3 work days in any 7 consecutive days*
- **Adjustment months:** February, March, August, September plus 7-day buffers before/after
- **Summer leave:** NIMS-recommended Obon period (4 weekdays around Aug 13-16, excluding weekends and holidays) is auto-marked off

## For administrators

1. Open `/admin/index_en.html`
2. Click `+ Add student` to create a new tab
3. Enter **Name** and **Email**, then set **Program**, **Start** date, and **Weekly pattern**
4. Click days on the calendar to fine-tune (monthly and annual totals update instantly)
5. Click `Send email` — your mail client opens with the recipient, subject, and share URL pre-filled
6. Or click `Copy link` to paste it into chat, docs, etc.

- Each student's data is stored independently in browser localStorage
- Click a tab to switch students; renaming updates the tab label instantly

## For students

1. Open the URL from the NIMS Junior office on any PC or smartphone
2. Your calendar, weekly pattern, and monthly totals appear
3. Use `Export CSV` to download data for Excel / Numbers
4. Editing is disabled — if updates are needed, ask the office to re-send the link

## Legend

| Color | Meaning |
|---|---|
| Blue | Work day |
| Light yellow | Adjustment period (freely editable in admin) |
| Light purple | Summer leave (Obon) |
| White | Off day |
| Red border + `!` | 3-day/week rule violation (weeks with holidays or summer leave are exempt) |

## Technical notes

- **Single HTML files** × 4 (admin JP/EN, student JP/EN), no build, no dependencies
- **State**: admin uses localStorage (`nims_junior_admin_v1`); student view is stateless and reads from URL fragment `#s=<base64>`
- **Email sending**: uses `mailto:` to open the administrator's default mail client (Outlook / Apple Mail / Gmail)
- **Holidays**: approximation formulas valid for 1980–2099 (includes equinoxes and substitute holidays)
